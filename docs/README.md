## Faculty Presentation Focus

For a short meeting, prioritize these talking points:

1. BullsEye is a privacy-aware AI grading workflow, not just a single grading prompt.
2. The current Lab 01 pilot shows systematic AI under-scoring compared with the human TA.
3. Calibration reduces MAE from 3.6 points to about 1.25 points.
4. The strongest publication gap is privacy-aware, rubric-calibrated, human-in-the-loop grading.
5. The next research steps are semantic RAG, model comparison, time-savings logging, and a larger dataset.
